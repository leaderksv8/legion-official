Оновлення сайту на GitHub (команди для PowerShell):

Крок 1: Перейдіть до папки з проєктом

cd D:\legion-of-titans


Крок 2: Додайте всі змінені файли до збереження Ця команда знаходить усі файли, які ви змінили, і готує їх до відправки.

git add .


Крок 3: Збережіть зміни з описом Замініть "Оновив контактну інформацію" на короткий опис того, що саме ви змінили. Це допоможе вам у майбутньому розуміти історію змін.

git commit -m "Оновив контактну інформацію"


Крок 4: Відправте оновлення на GitHub Ця команда завантажує ваші збережені зміни на сайт.

git push origin master
